void Categories::read_categories ( int  C,
int  K 
)

Lee la información de las categorías.

Precondición
C >= 1, K >= 4 En el canal de entrada hay primero C strings con los nombres asociados a las categorías identificadas por los valores de 1 a C en orden creciente. Luego, en el canal de entrada hay CxK enteros >= 0 que serán los puntos por categoría y nivel ordenados crecientemente por categorías y dentro de cada categoría, ordenados crecientemente por nivel.
Postcondición
Se borra el contenido de this->ctg_scores y reempleza con los puntos por categoría y nivel en el canal de entrada. Se borra el contenido de this->names y se reemplaza con los nombres en el canal de entrada.
